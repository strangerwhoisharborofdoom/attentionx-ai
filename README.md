# 🎯 AttentionX AI – Automated Content Repurposing Engine

## 🚀 Problem
Creators produce hours of long-form content, but audiences consume short-form. Extracting viral moments manually is slow, inefficient, and requires heavy editing effort.

## 💡 Solution
AttentionX AI automatically converts long videos into short, high-impact clips using AI-driven analysis.

## ✨ Key Features
- 📹 Upload long-form video
- 🔍 Detect high-impact emotional moments
- ✂️ Generate short clips
- 💬 AI-generated hook titles
- 📊 Viral Score Prediction System
- 🚀 Ready-to-share content

## 🧠 How It Works
1. Upload video
2. AI analyzes transcript + engagement signals
3. Identifies key moments
4. Generates clips with hooks & scores

## 🛠 Tech Stack

- Streamlit
- MoviePy (planned)
- Whisper (planned)
