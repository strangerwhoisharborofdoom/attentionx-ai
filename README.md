# 🎯 AttentionX AI – Automated Content Repurposing Engine

🚀 Turn 1 long video into multiple viral short clips automatically

---

## 🚀 Problem
Creators produce hours of long-form content (podcasts, lectures, interviews), but audiences consume short-form content. Extracting viral moments manually is slow, inefficient, and time-consuming.

---

## 💡 Solution
AttentionX AI automatically converts long videos into short, high-impact clips using AI-driven analysis.

---

## ✨ Key Features
- 📹 Upload long-form video
- 🔥 Detect high-impact emotional moments
- ✂️ Generate short clips
- 💬 AI-generated hook titles
- 📊 Viral Score Prediction System
- 🚀 Ready-to-share content

---

## 🧠 How It Works
1. Upload video  
2. AI analyzes content (simulated)  
3. Detects emotional peaks  
4. Generates clips + hooks  
5. Assigns viral score  

---

## 🧠 AI Pipeline
- 🎙 Transcription (Whisper - planned)
- 🔥 Emotional Peak Detection (audio + sentiment simulation)
- ✂️ Clip Extraction (MoviePy - planned)
- 💬 Hook Generation (LLM-inspired logic)
- 📊 Viral Score Prediction (custom scoring system)

---

## 🛠 Tech Stack
- Python
- Streamlit
- MoviePy (planned)
- Whisper (planned)
