# 🎯 AttentionX AI – Automated Content Repurposing Engine

## 🚀 Problem
Creators produce long-form videos, but audiences consume short-form content. Finding viral moments manually is slow.

## 💡 Solution
AttentionX AI converts long videos into viral short clips automatically.

## ✨ Features
- Upload video
- Generate clips
- Hook titles
- Viral score prediction

## 🛠 Tech Stack
- Python
- Streamlit
- MoviePy
- Whisper (planned)
